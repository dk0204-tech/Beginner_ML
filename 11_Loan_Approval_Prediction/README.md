# Loan Approval Prediction using Machine Learning

This project predicts whether a loan application will be approved using Machine Learning.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- SciPy

---

## Machine Learning Workflow

1. Load Dataset
2. Handle Missing Values
3. Exploratory Data Analysis
4. Data Preprocessing
5. Feature Selection
6. Train-Test Split
7. Train Classification Model
8. Evaluate Model
9. Predict Loan Approval

---

## Features Used

- Gender
- Married
- Dependents
- Education
- Self Employment
- Applicant Income
- Coapplicant Income
- Loan Amount
- Credit History

Target:
- Loan Status

---

## Model Used

Random Forest Classifier

---

## Visualizations

- Loan Status Distribution
- Gender vs Loan Status
- Education vs Loan Status

- Loan Amount Distribution
- Correlation Heatmap
- Confusion Matrix

---

## Evaluation Metrics

- Accuracy Score
- Classification Report
- Confusion Matrix

---

## Statistical Analysis

Performed Chi-Square Test between Gender and Loan Approval.

### Hypothesis

- H₀: Gender and loan approval are independent
- H₁: Gender and loan approval are related

---

## Key Insights

- Credit history strongly impacts approval
- Higher income increases approval probability
- Random Forest achieved strong classification accuracy
- Financial indicators play major roles

---

## Conclusion

The project demonstrates how Machine Learning can assist banks and financial institutions in automating loan approval decisions.
