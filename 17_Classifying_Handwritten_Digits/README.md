# Classifying Handwritten Digits using CNN

## Overview
This project uses Convolutional Neural Networks (CNNs)
to classify handwritten digits.

## Technologies Used
- TensorFlow
- Keras
- CNN
- Deep Learning

## Dataset
MNIST Handwritten Digits Dataset

## Features
- CNN architecture
- Image classification
- Accuracy/loss visualization
- Confusion matrix
- Digit prediction

## Accuracy
~99%
