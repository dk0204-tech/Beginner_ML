# Customer Segmentation using Machine Learning

This project performs Customer Segmentation using K-Means Clustering.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

---

## Machine Learning Workflow

1. Load Dataset
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Elbow Method
6. K-Means Clustering
7. Cluster Visualization
8. Customer Analysis

---

## Concepts Used

- Unsupervised Learning
- Clustering
- K-Means Algorithm
- Elbow Method
- Silhouette Score

---

## Features Used

- Age
- Annual Income
- Spending Score

## Visualizations

- Gender Distribution

- Customer Cluster Visualization

## Model Used

K-Means Clustering

---

## Evaluation Metric

- Silhouette Score

## Key Insights

- Premium customers were identified
- Spending behavior patterns were analyzed
- Customer groups help businesses improve marketing strategies
- K-Means effectively segmented customer groups

## Conclusion

This project demonstrates how unsupervised learning can be used to understand customer behavior and improve business decision-making.
