# Titanic Survival Prediction

A Machine Learning project predicting whether passengers survived the Titanic disaster.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

---

## Machine Learning Workflow

1. Load Dataset
2. Data Cleaning
3. Handle Missing Values
4. Feature Engineering
5. Data Visualization
6. Train Model
7. Evaluate Model
8. Make Predictions

---

## ML Model Used

Logistic Regression

---

## Features Used

- Passenger Class
- Gender
- Age
- Fare
- Family Members
- Embarkation Port

---

## Visualizations

- Survival Count
- Survival by Gender
- Passenger Class Analysis
- Age Distribution
- Confusion Matrix

---

## Accuracy

~80% to 85%

---

## Concepts Learned

- Binary Classification
- Data Cleaning
- Feature Engineering
- Encoding
- Logistic Regression
