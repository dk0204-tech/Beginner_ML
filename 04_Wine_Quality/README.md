# Wine Quality Prediction using Machine Learning

This project predicts wine quality using chemical properties and machine learning.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

---

## Machine Learning Workflow

1. Load Dataset
2. Explore Data
3. Visualize Features
4. Analyze Correlations
5. Train Classification Model
6. Evaluate Performance
7. Predict Wine Quality

---

## ML Model Used

Random Forest Classifier

---

## Visualizations

- Wine Quality Distribution
- Correlation Heatmap
- Alcohol vs Quality
- Acidity vs Quality
- Feature Importance Graph
- Confusion Matrix

---

## Accuracy

~85% to 92%
