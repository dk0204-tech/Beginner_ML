# Iris Flower Classification

A Machine Learning project that classifies Iris flower species using the Decision Tree Classifier.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

---

## Machine Learning Workflow

1. Load dataset
2. Explore data
3. Visualize patterns
4. Train ML model
5. Test model
6. Evaluate accuracy
7. Predict flower species

---

## Visualizations

- Species Count Plot
- Pairplot
- Correlation Heatmap
- Confusion Matrix

---

## Model Used

Decision Tree Classifier

---

## Accuracy

~100%

---

## Dataset Features

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## Flower Classes

- Setosa
- Versicolor
- Virginica
