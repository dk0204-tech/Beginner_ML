# Waiter's Tip Prediction using Machine Learning

This project predicts waiter tip amounts using restaurant billing data and Machine Learning.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- SciPy

---

## Machine Learning Workflow

1. Load Dataset
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Preprocessing
5. Feature Engineering
6. Train-Test Split
7. Train Linear Regression Model
8. Evaluate Model
9. Predict Tips

---

## Features Used

- Total Bill
- Gender
- Smoker
- Day
- Time
- Group Size

Target:
- Tip Amount

---

## Model Used

Linear Regression

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

---

## Statistical Analysis

Performed Pearson Correlation Test between total bill and tip amount.

### Hypothesis

- H₀: No significant relationship exists between total bill and tip
- H₁: Significant relationship exists between total bill and tip

The test showed statistically significant correlation.

---

## Key Insights

- Higher bills generally result in higher tips
- Dinner customers tend to tip more
- Group size impacts tipping behavior
- Total bill is the strongest predictor

---

## Conclusion

The Linear Regression model successfully predicts restaurant tip amounts with reasonable accuracy and demonstrates real-world business analytics applications.
