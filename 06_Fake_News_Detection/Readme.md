# Fake News Detection using NLP and Machine Learning

This project detects fake news articles using Natural Language Processing and Machine Learning.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- NLP
- TF-IDF
- WordCloud

---

## Machine Learning Workflow

1. Load Dataset
2. Clean Data
3. Analyze News Articles
4. Convert Text into Numerical Features
5. Train NLP Model
6. Evaluate Performance
7. Predict Fake News

## Statistical Analysis

Performed Independent T-Test to compare text lengths between fake and real news articles.

### Hypothesis
- H₀: Mean text lengths are equal
- H₁: Mean text lengths are different

The test showed statistically significant differences in article lengths.
