# Spam Detection using Machine Learning and NLP

This project classifies SMS messages as Spam or Ham using Natural Language Processing and Machine Learning.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- NLP
- WordCloud

---

## Machine Learning Workflow

1. Load Dataset
2. Clean Data
3. Explore Dataset
4. Convert Text to Numbers
5. Train NLP Model
6. Evaluate Performance
7. Predict Spam Messages

---

## NLP Concepts Used

- Text Vectorization
- Bag of Words
- CountVectorizer
- Word Frequency

---

## ML Model Used

Multinomial Naive Bayes

---

## Visualizations

- Spam vs Ham Distribution
- Message Length Analysis
- Spam Word Cloud
- Confusion Matrix

---

## Accuracy

~97% to 99%
